🔍 核心概念大解析 单项式×单项式 · 单项式×多项式 · 多项式×多项式
🔹 单项式 × 单项式 系数相乘，同底数幂相乘 3x²y · 2xy³ = 6x³y⁴
🔹 单项式 × 多项式 用单项式乘以多项式的每一项 a(b+c) = ab+ac
🔹 多项式 × 多项式 逐项相乘，再合并同类项 (a+b)(c+d) = ac+ad+bc+bd
🔹 乘法 交换律、结合律、分配律 都适用 🔹 结果要 合并同类项
🧠 数学锦囊 · 公式 核心法则
单项式×单项式
系数×系数 · 同底数幂相乘
单项式×多项式
m(a+b) = ma + mb
多项式×多项式
(a+b)(c+d) = ac+ad+bc+bd
注意
结果要 合并同类项
💡 口诀：单项乘单项，系数乘系数，同底幂相乘；单项乘多项，分配律来帮忙；多项式乘多项式，逐项相乘再合并。
✏️ 典型举例 步骤演示
例1 计算：3xy² · 2y³

解： 3xy² · 2y³ = (3×2) · x · (y²·y³) = 6xy⁵

系数 3×2=6 x 不变 y²·y³ = y⁵
例2 计算：(−4x²)(3x+1)

解： (−4x²)(3x+1) = (−4x²)·3x + (−4x²)·1 = −12x³ − 4x²

(−4x²)·3x = −12x³ (−4x²)·1 = −4x²
例3 计算：(a+3)(a−2)

解： (a+3)(a−2) = a·a + a·(−2) + 3·a + 3·(−2) = a² −2a+3a−6 = a² + a − 6

a·a = a² a·(−2) = −2a 3·a = 3a 3·(−2) = −6 合并：a²+a−6
⚠️ 易错题型 避坑指南
❌ 易错1：漏乘项

单项式乘以多项式时，要乘以每一项，不能漏项。
例：a(b+c) = ab+ac，不能写成 ab+c。

✗ a(b+c) = ab+c    ✓ a(b+c) = ab+ac
❌ 易错2：符号处理错误

负数的乘法要注意符号。
例：(−2x)·(3x) = −6x²，不是 6x²。

(−2x)·(3x) = −6x² ✓ ✗ (−2x)·(3x) = 6x²
❌ 易错3：多项式乘多项式漏项

(a+b)(c+d) = ac+ad+bc+bd，不能写成 ac+bd。

✗ (a+b)(c+d) = ac+bd    ✓ (a+b)(c+d) = ac+ad+bc+bd

📝 复习巩固题 小测验
1. 计算：4x²y · (−3xy³)
4x²y · (−3xy³) = 4×(−3) · x²·x · y·y³ = −12x³y⁴
系数 4×(−3)=−12 x²·x=x³ y·y³=y⁴
2. 计算：(x−3)(x+2)
(x−3)(x+2) = x·x + x·2 + (−3)·x + (−3)·2 = x² + 2x − 3x − 6 = x² − x − 6
x² 2x−3x=−x −6
