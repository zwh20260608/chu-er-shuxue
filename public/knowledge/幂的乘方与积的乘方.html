🔍 核心概念大解析 幂的乘方 · 积的乘方
📌 幂的乘方

(am)n = am×n

底数不变，指数相乘。

例：(103)5 = 1015

📌 积的乘方

(ab)n = an · bn

把积的每一个因式分别乘方，再把所得的幂相乘。

例：(2a)3 = 8a3

(am)n = amn
底数 不变 指数 相乘
(ab)n = an · bn
各因式 分别乘方 幂 相乘
🔹 幂的乘方：底数 不变，指数 相乘 🔹 积的乘方：各因式 分别乘方 🔹 注意：不要与同底数幂乘法混淆
🧠 数学锦囊 · 公式 核心法则
幂的乘方
(am)n = amn
积的乘方
(ab)n = anbn
推广
(abc)n = anbncn
易混淆
am·an = am+n ← 指数加
(am)n = amn ← 指数乘
💡 口诀：幂的乘方，底不变，指数乘；积的乘方，各因式分别乘方再相乘。
✏️ 典型举例 步骤演示
例1 计算：(103)5

解： (103)5 = 103×5 = 1015

底数 10 不变 指数 3×5=15
例2 计算：(2a)3

解： (2a)3 = 23 · a3 = 8a3

2 和 a 分别乘方 2³=8，a³=a³ 结果相乘：8a³
例3 计算：(−2x3y)4

解： (−2x3y)4 = (−2)4 · (x3)4 · y4 = 16 · x12 · y4 = 16x12y4

(−2)⁴=16 (x³)⁴=x¹² y⁴=y⁴
⚠️ 易错题型 避坑指南
❌ 易错1：混淆幂的乘方与同底数幂乘法

幂的乘方：指数相乘 (am)n = amn
同底数幂乘法：指数相加 am · an = am+n

✗ (a²)³ = a⁵    ✓ (a²)³ = a⁶    ✓ a²·a³ = a⁵
❌ 易错2：积的乘方漏乘系数或符号

积的乘方要每一项都乘方，包括系数和符号。
注意：(−a)n，当n为奇数时结果为负，n为偶数时为正。

(−2a)² = 4a² ✓ (−2a)³ = −8a³ ✓ ✗ (−2a)² = −4a²
❌ 易错3：指数相乘时忘记乘

(am)n = am×n，指数必须相乘，不能相加。

✗ (x²)³ = x⁵    ✓ (x²)³ = x⁶
