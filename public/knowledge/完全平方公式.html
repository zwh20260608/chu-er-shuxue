🧠 数学锦囊 · 公式 核心法则
完全平方公式
(a+b)² = a²+2ab+b²
(a−b)² = a²−2ab+b²
记忆口诀
首平方，尾平方
两倍首尾放中央
常见应用
99² = (100−1)²
= 10000−200+1=9801
易混淆
(a+b)² ≠ a²+b²
缺少 2ab 项
💡 口诀：首平方，尾平方，两倍首尾放中央；符号看前方，加减要分清。
✏️ 典型举例 步骤演示
例1 计算：(4m+n)²

解： (4m+n)² = (4m)² + 2·(4m)·n + n² = 16m² + 8mn + n²

首平方：(4m)²=16m² 2×首×尾：2×4m×n=8mn 尾平方：n²
例2 计算：(y − ½)²

解： (y − ½)² = y² − 2·y·½ + (½)² = y² − y + ¼

首平方：y² −2×首×尾：−2×y×½ = −y 尾平方：(½)²=¼
例3 计算：99²（利用公式简便计算）

解： 99² = (100−1)² = 100² − 2×100×1 + 1² = 10000 − 200 + 1 = 9801

a=100，b=1 100²=10000 −2×100×1=−200 +1
⚠️ 易错题型 避坑指南
❌ 易错1：漏掉中间项 2ab

(a+b)² = a² + 2ab + b²，不能写成 a² + b²。
中间项 2ab 是必须的！

✗ (x+3)² = x²+9    ✓ (x+3)² = x²+6x+9
❌ 易错2：符号处理错误

(a−b)² = a² − 2ab + b²，中间项是 减号。
注意：(−a−b)² = (a+b)²，因为负负得正。

(−2x+5)² = 4x²−20x+25 ✓ ✗ (−2x+5)² = 4x²+20x+25
❌ 易错3：混淆完全平方与平方差

(a+b)² = a²+2ab+b²   ≠   a²−b²
(a+b)(a−b) = a²−b² 才是平方差公式。

(x+3)² = x²+6x+9  |  (x+3)(x−3) = x²−9

📝 复习巩固题 小测验
1. 计算：(3x+2)²
(3x+2)² = (3x)² + 2·(3x)·2 + 2² = 9x² + 12x + 4
首平方：9x² 2×3x×2=12x 尾平方：4
2. 计算：102²（利用完全平方公式）
102² = (100+2)² = 100² + 2×100×2 + 2² = 10000 + 400 + 4 = 10404
a=100，b=2 100²+400+4=10404
