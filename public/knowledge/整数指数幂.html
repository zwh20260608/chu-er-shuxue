🔍 核心概念大解析 负指数 · 零指数 · 运算性质
指数从正整数扩展到0和负整数
… a-3 a-2 a-1 a0 a1 a2 a3 …
🔹 负指数：a-n = 1/an (a≠0) 🔹 零指数：a0 = 1 (a≠0)
负整数指数幂： a−n = 1an (a ≠ 0，n 为正整数)
零指数幂： a0 = 1 (a ≠ 0)
负指数表示倒数，零指数表示1
🔹 同底数幂相乘 am · an = am+n
🔹 幂的乘方 (am)n = amn
🔹 积的乘方 (ab)n = anbn
🔹 同底数幂相除 am ÷ an = am−n
🔹 商的乘方 (a/b)n = an/bn
🔹 负指数 a−n 是 an 的倒数 🔹 底数 a ≠ 0 🔹 所有运算性质对 全体整数 指数成立
🧠 数学锦囊 · 公式 核心结论
负指数定义
a−n = 1/an (a≠0)
零指数
a0 = 1 (a≠0)
运算法则
am·an = am+n
(am)n = amn
注意
底数 不能为0
负指数 → 倒数
💡 口诀：负指数，取倒数；零指数，等于1；运算性质全适用，底数不为零。
✏️ 典型举例 步骤演示
例1 计算：a−2 ÷ a5

解： a−2 ÷ a5 = a−2−5 = a−7 = 1/a7

同底数幂相除，指数相减 −2−5 = −7 a−7 = 1/a7
例2 计算：(b3/a2)−2

解： (b3/a2)−2 = b−6/a−4 = a4b−6 = a4/b6

商的乘方，分子分母分别乘方 负指数变正：a−4→a4，b6在分母
例3 计算：(a−1b2)3

解： (a−1b2)3 = a−3b6 = b6/a3

积的乘方，各因式分别乘方 (a−1)3 = a−3 (b2)3 = b6
⚠️ 易错题型 避坑指南
❌ 易错1：负指数与负数混淆

a−n 表示 倒数，不是负数。
例：2−3 = 1/8 = 0.125，不是 −8

✗ 2−3 = −8    ✓ 2−3 = 1/8
❌ 易错2：底数为0的情况

0 的负指数和0指数都 无意义。
0−2 = 1/0² = 1/0，分母为0，无意义。

✓ 20 = 1  |  ✗ 00 无意义  |  ✗ 0−2 无意义
❌ 易错3：运算性质使用错误

负指数也要遵循运算性质。
例：a−2 · a−3 = a−5 = 1/a5，不是 a6

✗ a−2·a−3 = a6    ✓ a−5 = 1/a5

📝 复习巩固题 小测验
1. 计算：3−2 + (−2)0
3−2 = 1/3² = 1/9，(−2)0 = 1
1/9 + 1 = 10/9
3−2=1/9 (−2)0=1 1/9+1=10/9
2. 计算：a−2b2 · (a2b−2)−3
(a2b−2)−3 = a−6b6
a−2b2 · a−6b6 = a−8b8 = b8/a8
