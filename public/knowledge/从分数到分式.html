🔍 核心概念大解析 分式定义 · 有意义 · 值为0
一般地，如果 A、B 表示两个整式，且 B 中含有字母，
那么式子 A/B 叫做分式
分式 A/B 中，A 叫做分子，B 叫做分母
🔹 分式有意义的条件 分母 B ≠ 0 例：1/(x−1) 有意义，则 x≠1
🔹 分式值为0的条件 分子 A = 0 且 分母 B ≠ 0 例：(x−2)/(x+3) = 0，则 x=2
🔹 分式与分数的联系 分数是 特殊 的分式（分母中不含字母） 分数：3/4  |  分式：a/b
🔹 分式 A/B 中，B 必须含有字母 🔹 分式有意义 ↔ 分母 ≠ 0 🔹 分式值为0 ↔ 分子 = 0 且 分母 ≠ 0
🧠 数学锦囊 · 公式 核心法则
分式定义
A/B（B中含字母）
有意义
B ≠ 0
值为0
A = 0 且 B ≠ 0
整式 vs 分式
分母不含字母 → 整式
分母含字母 → 分式
💡 口诀：分式形如A/B，B中含字母；分母不为零，分式有意义；分子零分母非零，分式值为零。
✏️ 典型举例 步骤演示
例1 下列式子中，哪些是分式？

1/x x/3 4/(3b³+5) (2a−5)/3

解： 分式有：1/x、4/(3b³+5)
判断依据：分母中 是否含有字母。

例2 分式 2/(3x) 有意义，x 应满足什么条件？

解： 分母 3x ≠ 0 → x ≠ 0

分母 3x 3x ≠ 0 x ≠ 0
例3 分式 (x−1)/(x+2) 的值为0，求x的值。

解： 分子 x−1 = 0 → x = 1
当 x=1 时，分母 x+2 = 3 ≠ 0
∴ x = 1

分子 = 0 检验分母 ≠ 0
⚠️ 易错题型 避坑指南
❌ 易错1：混淆整式与分式

判断关键是看分母 是否含有字母。
例：x/3 是整式（分母不含字母），3/x 是分式（分母含字母）。

✓ x/3 是整式    ✓ 3/x 是分式
❌ 易错2：分式值为0时，忽略分母不为0

分式值为0，必须 同时满足 分子=0 且 分母≠0。
例：(x²−4)/(x−2) 当 x=2 时，分子为0但分母也为0，分式无意义。

✗ x=2 时值为0    ✓ 无解（x≠2）
❌ 易错3：分式有意义条件理解错误

分式有意义只要求 分母 ≠ 0，与分子无关。
例：1/(x²+1) 中，x²+1 > 0 恒成立，所以 x 取任意实数 分式都有意义。

✓ x²+1 ≠ 0 恒成立 → x取任意实数
