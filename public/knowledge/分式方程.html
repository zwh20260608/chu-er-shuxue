📌 分式方程的定义

分母中含有未知数的方程叫做分式方程。

如：9030+v = 6030−v

📌 解分式方程的步骤
① 去分母 乘最简公分母
→
② 解整式方程 得解
→
③ 检验 代入最简公分母
→
✅ 写结论 原方程的解
⚠️ 增根

使最简公分母为0的根，不是原分式方程的解，必须舍去。

检验：将解代入最简公分母，若为0则为增根。

🔹 分式方程 必须检验 🔹 增根 不是 原方程的解 🔹 去分母时 不能漏乘 项
🧠 数学锦囊 · 公式 核心法则
解法步骤
① 去分母（乘最简公分母）
② 解整式方程
③ 检验（最简公分母≠0）
增根
使最简公分母 = 0
必须舍去
检验方法
代入最简公分母
≠ 0 → 原方程的解
= 0 → 增根，舍去
注意
去分母时
每一项都要乘
不能漏乘常数项
💡 口诀：分式方程要解好，去分母是关键；整式方程解出来，检验增根不能忘。
✏️ 典型举例 步骤演示
例1 解方程：2x−3 = 3x

解： 最简公分母 = x(x−3)
两边乘 x(x−3)：2x = 3x − 9
解得：x = 9
检验：当 x=9 时，x(x−3) ≠ 0
∴ 原方程的解为 x = 9

去分母 解整式方程 检验通过
例2 解方程：xx−1 − 1 = 3(x−1)(x+2)

解： 最简公分母 = (x−1)(x+2)
两边乘：x(x+2) − (x−1)(x+2) = 3
解得：x = 1
检验：当 x=1 时，(x−1)(x+2) = 0
∴ x=1 是增根，原方程无解

去分母 解得 x=1 检验：最简公分母=0 ❌ 增根，舍去
例3 列分式方程解应用题

问题：甲做90个零件与乙做60个零件所用时间相等，甲每小时比乙多做6个，求甲、乙每小时各做多少个？
解： 设乙每小时做 x 个，则甲每小时做 (x+6) 个
90x+6 = 60x
解得：x = 12
检验：x=12 时，x(x+6) ≠ 0
∴ 乙每小时做 12 个，甲每小时做 18 个

⚠️ 易错题型 避坑指南
❌ 易错1：去分母时漏乘常数项

例：xx−1 − 1 = 3(x−1)(x+2)
去分母时，−1 也要乘 (x−1)(x+2)！
✗ 只乘左边分式    ✓ 每一项都乘

✗ x = 3    ✓ x(x+2) − (x−1)(x+2) = 3
❌ 易错2：忘记检验增根

解分式方程 必须检验！
例：1x−5 = 10x²−25
解得 x=5，但 x=5 使分母为0，是增根，必须舍去。

✗ 直接写 x=5    ✓ 检验后舍去，无解
❌ 易错3：最简公分母找错

例：3x²+2x − 1x²−2x = 0
分解：x²+2x = x(x+2)，x²−2x = x(x−2)
最简公分母 = x(x+2)(x−2)

✓ x(x+2)(x−2)    ✗ x(x²−4) 不够完整

📝 复习巩固题 小测验
1. 解方程：5x = 7x−2
最简公分母 = x(x−2)
5(x−2) = 7x → 5x−10 = 7x → −2x = 10 → x = −5
检验：x=−5 时，x(x−2) ≠ 0
∴ 原方程的解为 x = −5
去分母 解整式方程 检验通过
2. 解方程：2x−1 = 4x²−1
x²−1 = (x−1)(x+1)，最简公分母 = (x−1)(x+1)
2(x+1) = 4 → 2x+2=4 → x=1
检验：x=1 时，(x−1)(x+1) = 0
∴ x=1 是增根，原方程无解
去分母 解得 x=1 增根，无解
