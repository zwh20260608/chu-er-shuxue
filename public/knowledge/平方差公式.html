🔍 核心概念大解析 平方差公式 · 结构 · 几何意义
(a + b) · (a − b) = a2 − b2
两个数的和与这两个数的差的积，等于这两个数的平方差。
📐 面积图：a² − b² = (a−b)(a+b)
🔹 结构特征 左边：两项 相同，一项 相反 (▲ + ■)(▲ − ■)
🔹 结果特征 右边：平方差（两项平方相减） ▲² − ■²
🔹 公式中的 a 和 b 可以是数、字母或整式 🔹 注意：a² − b² 是平方差，不是 (a−b)²
🧠 数学锦囊 · 公式 核心法则
平方差公式
(a+b)(a−b) = a²−b²
识别方法
找相同的项 ▲ 和相反的项 ■
常见应用
102×98 = (100+2)(100−2)
= 100²−2² = 9996
易混淆
(a−b)² = a²−2ab+b²
≠ a²−b²
💡 口诀：两数和乘两数差，等于两数平方差；结构相同找相反，平方相减记心间。
✏️ 典型举例 步骤演示
例1 计算：(3x+2)(3x−2)

解： (3x+2)(3x−2) = (3x)² − 2² = 9x² − 4

a = 3x，b = 2 (3x)² = 9x² 2² = 4
例2 计算：102 × 98

解： 102×98 = (100+2)(100−2) = 100² − 2² = 10000 − 4 = 9996

a = 100，b = 2 100² = 10000 2² = 4
例3 计算：(−x+2y)(−x−2y)

解： (−x+2y)(−x−2y) = (−x)² − (2y)² = x² − 4y²

a = −x，b = 2y (−x)² = x² (2y)² = 4y²
⚠️ 易错题型 避坑指南
❌ 易错1：混淆 (a+b)(a−b) 与 (a−b)²

(a+b)(a−b) = a²−b²   ≠   a²−2ab+b²
(a−b)² = a²−2ab+b²

✗ (x+3)(x−3) = x²−6x+9    ✓ (x+3)(x−3) = x²−9
❌ 易错2：没有找到"相同的项"和"相反的项"

平方差公式要求两项中 一项相同，一项相反。
例：(a+b)(a+b) 是 完全平方，不是平方差。

(a+b)(a−b) ✓ 平方差 (a+b)(a+b) ✗ 完全平方
❌ 易错3：忘记括号或符号错误

例：(−2x+3)(−2x−3) = (−2x)² − 3² = 4x² − 9
注意：不能写成 −4x² − 9 或 4x² + 9。

✓ (−2x+3)(−2x−3) = 4x²−9    ✗ = −4x²−9
