💡 口诀：全等三角形，对应顶点要写对；对应边角都相等，性质记心间。
✏️ 典型举例 步骤演示
例1 如图，△ABC ≌ △BAD，点A和点B，点C和点D是对应顶点，∠BAC=65°，∠ABC=26°，求∠CBD。

解：∵ △ABC ≌ △BAD
∴ ∠ABD = ∠BAC = 65°（对应角相等）
∠CBD = ∠ABD − ∠ABC = 65° − 26° = 39°

例2 如图，△ABC ≌ △DEF，AB=5，BC=7，∠A=40°，求DE、EF和∠D。

解：∵ 全等三角形对应边相等，对应角相等
∴ DE = AB = 5，EF = BC = 7，∠D = ∠A = 40°

⚠️ 易错题型 避坑指南
❌ 易错1：对应顶点写错位置

△ABC ≌ △DEF 表示 A↔D，B↔E，C↔F。
常见错误：把 △ABC ≌ △DEF 写成 △ABC ≌ △DFE（对应顶点错位）。

❌ 易错2：认为全等三角形面积相等但周长不一定相等

全等三角形 完全重合，所以 周长相等，面积也相等。

1. 如图，△ABC ≌ △DEF，∠A = 50°，∠B = 70°，AB = 4，DF = 6，求∠F 和 AC 的长。
∠C = 180° − 50° − 70° = 60°
∵ △ABC ≌ △DEF ∴ ∠F = ∠C = 60°
AC = DF = 6
2. 如图，△ABC ≌ △BAD，∠BAC = 40°，∠ABC = 60°，求∠ABD 和 ∠ADB。
∵ △ABC ≌ △BAD，A↔B，C↔D
∴ ∠ABD = ∠BAC = 40°
∠ADB = ∠ACB = 180° − 40° − 60° = 80°
