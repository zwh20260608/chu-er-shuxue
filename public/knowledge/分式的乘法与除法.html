分式乘法：分子乘分子，分母乘分母  |  分式除法：除以一个分式等于乘以它的倒数

① 除法变乘法
② 分解因式

💡 口诀：分式乘法分子乘，分母乘；除法变乘法，乘倒数；结果要化简，最简分式。
✏️ 典型举例 步骤演示
例1 计算：4x3y · y2x³

解： 4x3y · y2x³ = 4xy6x³y = 23x²

分子×分子：4x·y=4xy 分母×分母：3y·2x³=6x³y 约分：4xy6x³y = 23x²
例2 计算：ab³2c² ÷ −5a²b²4cd

解： ab³2c² ÷ −5a²b²4cd = ab³2c² · 4cd−5a²b² = 4ab³cd−10a²b²c² = −2bd5ac

除法变乘法，取倒数 约分：4ab³cd−10a²b²c² = −2bd5ac
例3 计算：( −2a²b3c )²

解： ( −2a²b3c )² = (−2a²b)²(3c)² = 4a⁴b²9c²

分子分母分别乘方 (−2a²b)² = 4a⁴b² (3c)² = 9c²
⚠️ 易错题型 避坑指南
❌ 易错1：除法没有变乘法

分式除法要 乘以除式的倒数，不能直接分母相除。
例：ab ÷ cd = ab · dc

✗ ab ÷ cd = a·cb·d    ✓ 乘倒数
❌ 易错2：约分不彻底

运算结果要化成 最简分式，分子分母不能再有公因式。
例：4x²y6xy² 应约分为 2x3y

✗ 4x²y6xy²    ✓ 2x3y
❌ 易错3：符号处理错误

负号的幂要注意。
例：( −2xy )³ = (−2x)³y³ = −8x³y³ （指数为奇数，结果为负）

✓ (−2x)³ = −8x³  |  ✓ (−2x)⁴ = 16x⁴
