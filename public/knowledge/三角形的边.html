✏️ 典型例题 方法·步骤
例1 判断 3, 4, 8 能否组成三角形？

解：3+4=7 < 8，不能组成三角形。

例2 等腰△周长为18，一边长4，求其他两边。

解：① 4为底 → 腰=(18−4)/2=7，三边7,7,4 ✓
② 4为腰 → 底=18−8=10，三边4,4,10 ✗ (4+4<10)
∴ 其他两边为 7, 7

⚠️ 易错题型 避坑指南
❌ 易错1：只检查一组两边之和

判断 2, 5, 6：有人只算 2+5=7>6 就认为可以，但必须验证 所有 组合！

✅ 正解：2+5=7>6 ✓，2+6=8>5 ✓，5+6=11>2 ✓，可以组成

❌ 易错2：等腰△分类讨论不验证

等腰△周长20，一边长6，求其他两边。

⚠️ 注意：6为腰 → 底=8，三边6,6,8 ✓；
6为底 → 腰=7，三边6,7,7 ✓。
两种情况都成立！

📝 复习巩固题 小测验
1. 下列各组线段中，能组成三角形的是（ ）
A. 2, 3, 5   B. 4, 5, 9   C. 3, 4, 6   D. 1, 2, 4
✅ 答案：C（3+4=7 > 6，且任意两边之和都大于第三边）
2. 已知三角形的两边长分别为 5 和 8，则第三边 c 的取值范围是 ______。
由 |8−5| < c < 8+5，得 3 < c < 13
3. 等腰三角形的一边长为 5，另一边长为 9，则它的周长是 ______。
分类讨论：
① 5为腰 → 三边 5,5,9，5+5=10>9 ✓，周长 = 19
② 9为腰 → 三边 9,9,5，5+9>9 ✓，周长 = 23
∴ 周长为 19 或 23
4. 如图，在△ABC中，AB=AC，周长为16，且AB比BC大2，求三边长。
设 BC=x，则 AB=AC=x+2。
周长：x+2 + x+2 + x = 16 → 3x+4=16 → x=4。
∴ 三边为 AB=AC=6，BC=4。
