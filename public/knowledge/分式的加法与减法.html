🔍 核心概念大解析 同分母 · 异分母 · 通分
同分母加减： ac ± bc = a ± bc
异分母加减： ab ± cd = ad ± bcbd
同分母：分母不变，分子相加减  |  异分母：先通分，再加减
🔹 同分母加减 分母不变，分子相加减 5x+3yx²−y² − 2xx²−y² = 3x+3yx²−y² = 3x−y
🔹 异分母加减 通分 → 同分母 → 加减 12 + 13 = 36 + 26 = 56
🔹 最简公分母 各分母因式的 最高次幂 的积 对于 1a²−4 和 1a²−4a+4 ，最简公分母 = (a+2)(a−2)²
🔹 结果要 约分 到最简分式 🔹 异分母通分时，最简公分母要 取全 🔹 分子相加减时注意 括号 和 符号
🧠 数学锦囊 · 公式 核心法则
同分母加减
ac ± bc = a±bc
异分母加减
通分 → 同分母加减
找最简公分母
最简公分母
各分母因式
最高次幂的积
运算步骤
① 判断同/异分母
② 通分（若需要）
③ 加减并化简
💡 口诀：同分母，分母不变分子加减；异分母，先通分再加减；结果要化简，最简分式。
✏️ 典型举例 步骤演示
例1 计算：5x+3yx²−y² − 2xx²−y²

解： 5x+3yx²−y² − 2xx²−y² = 5x+3y−2xx²−y² = 3x+3y(x+y)(x−y) = 3x−y

同分母，分母不变 分子相减：5x+3y−2x=3x+3y 约分：3(x+y)/(x+y)(x−y) = 3/(x−y)
例2 计算：12m²−9 + 23−m

解： 最简公分母 = (m+3)(m−3)
12(m+3)(m−3) − 2m−3 = 12−2(m+3)(m+3)(m−3) = 12−2m−6(m+3)(m−3) = −2m+6(m+3)(m−3) = −2m+3 = −2m+3

分解分母：m²−9=(m+3)(m−3) 3−m = −(m−3) 通分后加减 约分得 −2/(m+3)
例3 计算：56ab − 23ac + 34abc

解： 最简公分母 = 12abc
5·2c12abc − 2·4b12abc + 3·312abc = 10c−8b+912abc = 10c−8b+912abc

最简公分母 12abc 分别通分 分子相加减
⚠️ 易错题型 避坑指南
❌ 易错1：异分母直接加减分子

异分母分式不能直接加减分子，必须 先通分！
例：1a + 1b ≠ 2a+b    ✓ 应通分为 a+bab

✗ 1a + 1b = 2a+b    ✓ a+bab
❌ 易错2：通分时漏乘项

通分时，每个分子要乘以 最简公分母 ÷ 原分母 的因式。
例：1a²−4 通分到 (a+2)(a−2)²，分子要乘 (a−2)

✗ 1a²−4 → 1(a+2)(a−2)²    ✓ 分子乘 (a−2)
❌ 易错3：符号处理错误

减法时，减去一个分式 要加括号，注意符号变化。
例：ab − cd = ad−bcbd ，分子是 ad − bc，不是 ad + bc

✓ ab − cd = ad−bcbd
