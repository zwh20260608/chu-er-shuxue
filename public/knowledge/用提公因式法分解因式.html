🔍 核心概念大解析 公因式 · 提公因式法
pa + pb + pc = p(a + b + c)
把多项式各项的 公因式 提取出来，写成公因式与另一个因式乘积的形式
🔹 公因式 多项式各项都含有的 公共 因式 系数取最大公约数 · 字母取最低次幂
🔹 提公因式法 将公因式提到括号外面，多项式除以公因式得到另一个因式
🔹 因式分解与整式乘法是 互逆 变形 🔹 要分解到 不能再分解 为止
🧠 数学锦囊 · 公式 核心法则
提公因式法
pa + pb + pc = p(a+b+c)
找公因式
系数：最大公约数
字母：相同字母最低次幂
注意
提取公因式后，括号内
项数 不变
检验
用整式乘法验证
结果是否等于原多项式
💡 口诀：公因式，要提净；系数最大公约数，字母最低次幂；提出公因剩因式，乘法验证保正确。
✏️ 典型举例 步骤演示
例1 分解因式：8a³b² + 12ab³c

解： 公因式 = 4ab²
8a³b² + 12ab³c = 4ab²·(2a²) + 4ab²·(3bc) = 4ab²(2a² + 3bc)

系数 8和12 → 最大公约数 4 a：最低次 a¹ b：最低次 b²
例2 分解因式：2a(b+c) − 3(b+c)

解： 公因式 = (b+c)
2a(b+c) − 3(b+c) = (b+c)(2a − 3) = (b+c)(2a−3)

公因式是 (b+c) 提取后剩 2a 和 −3
例3 分解因式：4(a−b)³ + 8(b−a)²

解： (b−a)² = (a−b)²
4(a−b)³ + 8(a−b)² = 4(a−b)²·(a−b) + 4(a−b)²·2
= 4(a−b)²(a−b+2)

公因式 4(a−b)² 注意 (b−a)² = (a−b)²
⚠️ 易错题型 避坑指南
❌ 易错1：漏提公因式或提不彻底

例：8a³b² + 12ab³c 只提 4ab，剩下 2a²b + 3b²c 还有公因式 b。
正确：应提 4ab²。

✗ 4ab(2a²b+3b²c)    ✓ 4ab²(2a²+3bc)
❌ 易错2：括号内漏项

提取公因式后，括号内的项数应与原多项式相同。
例：3x² − 4xy² + x 提 x 后，不能写成 x(3x − 4y²)。

✗ x(3x − 4y²)    ✓ x(3x − 4y² + 1)
❌ 易错3：符号处理错误

当首项系数为负时，提公因式要注意符号。
例：−2x + 3x² 提 x 得 x(−2 + 3x)，不能写成 −x(2 − 3x) 虽然等价，但要注意统一。

−2x+3x² = x(−2+3x) ✓ = −x(2−3x) ✓ 等价

📝 复习巩固题 小测验
1. 分解因式：6a²b + 9ab² − 15ab
公因式 = 3ab
6a²b + 9ab² − 15ab = 3ab(2a + 3b − 5)
答案：3ab(2a+3b−5)
系数最大公约数 3 a最低次 a¹ b最低次 b¹
2. 分解因式：2m(x−y) − 3n(x−y)
公因式 = (x−y)
2m(x−y) − 3n(x−y) = (x−y)(2m − 3n)
答案：(x−y)(2m−3n)
公因式是 (x−y) 提取后剩 2m 和 −3n
